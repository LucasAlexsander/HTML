    HTML

    HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.

    Resumo de HTML

    Abreviações:

    < b > = bold text < strong > = Important text

    < i > italic = Italic text | < em > = Emphasized text
    < mark > = Marked text
    < small > = Smaller text | < del > = Deleted text
    < ins > = Inserted text | < sub > = Subscript text
    < sup > = Superscript text
    < br > = Broke the line
    < hr > = Used to separate content, and is most often displayed as a horizontal rule
    < pre > = Displayed in a fixed-width font (usually Courier), and it preserves both spaces and line breaks

    Citações em HTML

    < blockquote > = Usado para citar um site, deixando o texto um pouco no meio

    < q > / Quotation Marks = Seria as aspas, e deixa o texto um pouco menor
    < abbr > / Abbriviation = Abreviação, usado em siglas. Quando passa o mouse em cima aparece o significado
    < address > = Usado para deixar informações do autor, como nome, número, etc
    < cite > = Destacar uma obra, deixando o nome dela em italico
    < bdo > / Bi-Directional Override = Se o site suportar faz com que ele deixe as letras de traz pra frente
    Para adicionar dois atributos as tags = Usa-se o ";" depois do primeiro atributo;

    Adicionando cores as coisas

    Podemos adicionar cor a quase tudo, nas letras, no background, nas bordas, etc.
    Existe 6 tipos de códigos que podem ser usados para adicionar essas cores, podendo ser a cor em sí mas inglês,
    RGB (red, blue, green), RGBA (red, blue, green, alpha), HEX, HSl:
    Para adicionarmos cor nas letras basta colocar style= "color: code" isso aparecerá a cor.
    Exemplo de cada um:
    Nome : style="color: red"
    RGB : style="color: (255, 0, 0)"
    RGBA : style="color: (255, 0, 0, 255)"
    HEX : style="color: #FF0000" | Onde varia de 0-255, 00-FF, sendo rr(red), gg(gree), bb(blue)
    HSL : style="color: (0, 100%, 50%)"

    Os códigos das cores ficam fixos em todos os casos, mudando somente onde ele irá ser aplicado.
    Exemplos:
    background-color: : Para background
    border: tamanho tipo cor : Para as bordas
    style="color: " : Para as letras

    O que é CSS

    O Cascading Style Sheets (CSS) é uma "folha de estilo" composta por ?camadas? e utilizada para definir a apresentação (aparência) em páginas da internet que adotam para o seu desenvolvimento linguagens de marcação (como XML, HTML e XHTML). Podendo ser usado para controlar a cor, fonte, tamanho do texto, espaço entre os elementos, como os elesmentos irão aparecer no layout, qual imagem vai aparecer no background ou a cor que será usada, e várias outras opções.
    Inline usamos style para atribuir valores;
    Internal usamos < style > dentro do elemento < head >;
    External usamos < link > elemento para link para o arquivo CSS.

    Inline CSS

    Inline pode ser usando tanto no < p > ou no < h1 >.
    Internal usando dentro do elemento < head > criando uma aba < style >, onde você pode criar tags e adicionar atributos.
    External usado também dentro do elemento < head > onde você cria o < link > e adicionará dentro dele o rel="stylesheet" e logo em seguida o href="nome_do_arquivo" para importar o arquivo com o CSS-Style.

    CSS Colors, Fonts and Size

    CSS color propriedade de texto para cor.
    O CSS font-family: propriedade para alterar a fonte.
    O CSS font-size: px propriedade para definir o tamanho da fonte.

    CSS Border

    Border defini a borda dos elementos do CSS.
    Padding: px usado juntamente com border para definir o tamanho entre o texto e a borda.
    Margin: px outro elemento que será usado com o border, para definir o espaço fora da borda.

    Link para CSS externo

    Style Sheets pode ser usado como referenciado com URL completa ou com o path relativo para uma link da web.
    Exemplo:
    < link rel="stylesheet" href="https://www.w3schools.com/html/styles.css" >;
    < link rel="stylesheet" href="/html/styles.css" >;
    < link rel="stylesheet" href="styles.css" >
