HTML links - Hyperlinks

HTML links são hyperlinks, onde se você clicar será direcionado a outra página, podendo abrir outra guia ou não.
Um link não tem um texto, ele pode ser tanto uma imagem ou um texto do HTML.

HTML link - Sintaxe

A tag < a > é um hyperlink. Onde você adicionar o link nela, segue o exemplo abaixo:
< a href= "url" > Texto de link < /a>
Onde o mais importante do atributo < a> é o atributo href, pois nele que você adicionará o link.
A parte onde o link vai estar será destacado no texto, podendo ficar com uma cor diferente, um traço embaixo e em itálico.
E clicando no texto você será direcionado a página escolhida.
< a href= "www.google"> Ir para o Google < /a >

HTML link - atributo target

Este atributo target definirá como o link será aberto, podendo ser aberto em outro guia, no mesmo. Abaixo irei dar mais exemplos.
_self : abrirá na mesma guia, o padrão
_blank : será aberto em uma nova guia
_parant : abre o documento na moldura principal, parecido com o _self
_top : abre o documento completo na janela

URL absoluta vs URL relativa

URL absoluta seria o link da url completa no atributo href de um outro website.
Já a URL relativa é um link do mesmo website, como por exemplo, ir em um novo guia do mesmo site.

HTML link - Usando uma imagem para acessar

Vamos precisar usar a tag < img > dentro da tag < a > , como no exemplo a seguir;

 < a href="google.com"> 
            < img src="www.imagem_qualquer.com" alt="Qualquer Imagem" > < /a >
Feito isso o link está na imagem.

Link para um endereço de e-mail

Você terá que usar a tag mailto: dentro do href para gerar um link para seu e-mail.
< a href="mailto:seu_e-mail@exemplo.com" > Link para e-mail < /a >

Botão para link

Para adicionarmos um botão ao seu site, precisamos colocar alguns comenado no JavaScript, ele faz com que você possa criar alguns eventos em seu site, como apertar um botão.
Um exemplo de como usariamos o código no HTML:
< button onclick="document.location='defaut.asp'">HTML Tutorial< /button>

Título de link

Neste comenado você colocara alguma informação quando a pessoa passa o mouse em cima do link, podendo falar pra onde ela esta indo.
Para usarmos o comando title: primeiro precisando deixar ele dentro do < a >. Siga o exemplo abaixo:
< a href="URL" title="Ir para a URL"> Link < /a >

Links Color

No HTML tem diferentes cores dependendo do que você fez, se você entrou no site vai ter um, se não vai ter outra e por ae vai.
Se você entrou no site vai ficar sublinhada e roxo;
Se você não entrou vai ficar sublinhada e azul;
Se o site está ativo vai ficar sublinhada e vermelho.
Tem alguns comando que você consegue alterar o texto que contém o link, sendo eles, link:, hover:, active: e visited:, esses são comando que você pode alterar, conforme o usúario tenha entrado ou não neles.
Significado:
link: : o link por padrão;
hover: : quando passa o mouse emcima;
active: : quando o link está ativo;
visited: : quando o link já foi acessado.
Usando o CSS conseguimos criar um botão, mas precisa usar alguns comenados dentro do comando de preferência passado anteriomente. Para isso, vamos escolher um background, uma cor do texto, a borda, onde o texto vai estar.
Um exemplo de um botão criado pelo CSS:

                          < style >
                            a:link, a:visited {
                                background-color: #f44336;
                                color: white;
                                padding: 15px 25px;
                                text-align: center;
                                text-decoration: none;
                                display: inline-block;
                            }

                            a:hover, a:active {
                                background-color: red;
                            }
                        < /style >

Criando BookMark

O BookMark é usado em site grande, onde podemos clicar em um texto e ser direcionado para ele.
Primeiro, precisamos criar um atributo id para o BookMark.
< h2 id="C4"> Capítulo 4 < /hr >
Segundo, vamos adcionar um link para o BookMark ("Pular para tal capítulo").
< a href="#C4"> Pular para o capítulo 4 < /a >
Terceiro(opcional), caso você queira tem como você criar um BookMark para outra página.
< a href="html_demo.html#C4"> Pular para o caítulo 4
