HTML Imagens Sintaxe

Para adcionarmos uma imagem ao nosso site precisamos usar o atributo < img >, ele fará com que o programa aceite imagens, logo depois, usaremos src (source) que nada mais é que o path (caminho) do arquivo, dentro do atributo < img >, feito isso você já terá uma imagem.
Caso você queria dar um título para imagem, precisamos colocar um title logo em seguida do src.
Usamos o alt caso você esteja usando um link do google, pois caso o link é perdido ou pare de funcionar o browser pode procurar uma parecida para substituir, mas sempre é recomedavem você exportar a imagem para o site.
Podemos mexer no tamanho da imagem usando wight:'num_px' para largura e height:'num_px' para altura.
Se você tiver as imagens em outra pasta você faz a mesma coisa que o link, coloca o path (caminho) dentro do src.
Para imagens em outro site, você coloca a URL completa no src e no alt: você coloca o dominio.
Tudo que foi dito funciona também para imagens animadas.
Imagens com links você vai usar o atributo < a > e colocará o link no href e o arquivo no src.
Por último mas não menos importante, o float, ele serve para colocar a imagem a direita ou a esquerda do texto.

Resumo de Imagem


HTML Image Maps

A tag < map > defini um mapa da imagem, podendo clicar em determinadas áreas. E as áeras são definidas através da tag < area >.
Exemplo: 

       < img src="workplace.jpg" alt="Workplace" usemap="#workmap" >

        < map name="workmap" >
            < area shape="rect" coords="34,44,270,350" alt="Computer" href="computer.htm" >
            < area shape="rect" coords="290,172,333,250" alt="Phone" href="phone.htm" >
            < area shape="circle" coords="337,300,44" alt="Coffee" href="coffee.htm" >
        < /map >
A imagem é importada através da < img >, mas o que muda este tipo de imagem para as outras é o atributo usemap.
O usempa começa com hash tag ('#') e logo em seguida vem o nome do arquivo, isso é usado para criar um relacionamento entre a imagem e o mapa.

Criando um Image Map

Então, adicionamos a tag < map >, usado para criar um Image Map, juntamente usamos o atributo nome, que tem praticamente a mesma função que o usemap.

Criando as Áreas

Para colocarmos a função de clicar em determinada área da imagem, vamos colocar a tag < area >.

Corpo

Shape="NOME"
Você pode definir a área de várias formas, sendo eles, redonda poligonal, retangular, ou o padrão.
rect - defines a rectangular region, x-axis and y-axis;
circle - defines a circular region, you put the locale and radius of the circle;
poly - defines a polygonal region, x-axis and y-axis but so must;
default - defines the entire region

JavaScript and Image Map

Exemplo:
                < map name="workmap">
                    < area shape="circle" coords="337,300,44" href="coffee.htm" onclick="myFunction()" >
                < /map >

                < script >
                    function myFunction() {
                        alert("You clicked the coffee cup!");
                }
                < /script >

Image Tag


Picture tag

O < picture > dá ao desenvolvedor mais flexibilidade ao importar determinada imagem, pois você consegue definir para quais dispositivos, com base na sua tela, irá aparecer determinada imagem.
< picture > é usado juntamente com o elemento < source > para diferentes imagens junto com o srcset que definira qual imagem irá aparecer com base na media. Media será a condição para que a imagem apareceça, escolhendo o seu tamanho mínimo.
Exemplo:
                < picture >
                < source media="(min-width: 650px)" srcset="img_food.jpg" >
                < source media="(min-width: 465px)" srcset="img_car.jpg" >
                < img src="img_girl.jpg" >
                < /picture >

Definição

Uma tabela em HTML consistem em células que ficam dentro de colunas e linhas. E para usar precisamos inserir o < table >.

Cabeçalhos de Tabela

Se você deseja que suas células sejam cabeçalhos, neste caso ira usar < th > ao invés de < td >

Dicas:

•Para fazermos uma coluna parecer duas vezes com o mesmo nome, ou dado, basta colocar colspan="quantidade_de_vezes".
•Para fazermos uma linha aparecer duas vezes ou mais com o mesmo dado, basta colocar rowspan="quantidade_de_vezes".

HTML - Tamanho das tabelas

As tabelas podem ter vários tipos de tamanhos para colunas, linhas ou tabelas inteiras.

Use o atributo style com as propriedades width ou height para especificar o tamanho da tabela, colunas ou linhas.

Largura da tabela

Para definir o tamanho de um tabela, adicione o atributo style ao elemento <table>:
Usar uma porcentagem como unidade de tamanho para uma largura significa a largura desse elemento em comparação com elemento pai.

Altura da linha da tabela

Para definir a altura de uma linha, basta inserir o style ao elemento da linha na tabela, e isto serve para coluna também.

HTML - Cabeçalho

As tabelas em HTML podem ter cabeçalhos para cada coluna ou linhas, ou para muitas colunas/linhas. Os cabeçalhos são definidor por th, onde cada th representa o cabeçalho na tabela.

Cabeçalhos na vertical

Para usar a primeira coluna como cabeçalho, defina a primeira célula em cada linha como um elemento th.

Alinhar cabeçalhos

Por padrão, os cabeçalhos das tabelas já estão em negrito e centralizados.

Para alinharmos eles à esquerda ou direita, usamos o text-align:, basta adicionarmos ao atributo style.

Cabeçalhos para várias colunas

Você pode ter um cabeçalho que se estende por mais de uma coluna. Para fazermos isso, precisamos colocar o atributo colspan no elemento th:

Legenda da Tabela

No HTML temos a opção de adcionarmos uma legenda a nossa tabela, podendedo ser um título também, para isso usamos o a tag <caption>. Ela deve ser inserida imediatamente após <table>

Preenchimento e espaçamento da tabela

Podemos ajudar o preenchimento dentro das células e também o espaço entre as células.

      

O preenchimento é o espço entre às células da tabela, usa-se a propriedade padding. Para mexermos só no preenchimento de cima usamos a propriedade padding-top, caso queira usas as outras posições usa-se as propriedade padding-bottom, padding-left e padding-right.

Espaçamento de células

Por padrão, o valor vem 2 pixels, mas podemos altera-lo.
Para isso, usamos a propriedade border-space no elemento <table>.

Colspan & Rowpan

Podem estender as células por várias linhas/colunas.

Para fazer a célula se estender por várias colunas usamos o colspan.
Para fazer a célula se estender por várias linhas usamos o rowspan.

Estilo da tabela

Lista de zebra

Se você adicionar uma cor de fundo a cada linha da tabela, obterá um belo efeito de listas de zebra.
Para definir o estilo de todos os outros elementos da tabela, use o :nth-child(even) use desta forma:
td:nth-child(even){
background-color: green;
}
Podemos usa-lo tanto no th quanto no td. Você pode combinar linhas verticais com as horizontais, e se usar uma cor transparente, obterá um resultado de sobreposição.
Use o rgba() para espeficiar a transparencia da cor.

Divisão horizontal

Caso você queria somente uma linha na parte inferior da sua tabela, basta adicionar a propriedade border-bottom em todos os elementos tr.

Animação ao passar o mouse em cima da tabela

Usamos o :hover no tr e escolhemos um background-color diferente ou aumentamos o tamanho da fonte, assim, damos uma animação simples para a tabela.

HTML - Colgroup

O elemento <colgroup> é usado para definir o estilo de colunas específicas de uma tabela.
Se você quiser estilizar as duas primeiras colunas de uma tabela, use os elementos <colgroup> e <col>.
O <colgroup> é um elemento que deve ser usado como um contêiner para as especificações da coluna.
Cada grupo é espeficico com um elemento <col>.
O atributo span espeficica quantas colunas obtêm o estilo.
O style espeficica o estilo a ser dado às colunas. O elemento <colgroup> deve ser colocado após o <caption> (caso tenha). Se você deseja usar mais colunas com estilos diferentes, use o <col> dentro do <colgroup>.

Colunas vázias

Se quiser estilizar colunas no meio de uma tabela, insira o elemento <col> "vázio" (sem estilos) para as colunas anteriores.

Esconder Colunas

Você pode esconder colunas usando a propriedade visibility: collapse.
