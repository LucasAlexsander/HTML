Listas em HTML

A lista ajuda aos desenvolvedores web a agruparem e organizarem um conjunto de itens em listas.
Podemos organizar as listas em dois tipos, as organizadas e as não organizadas.

Não organizadas

Para fazermos este tipo de lista primeiro temos que usar a tag <ul>. Cada item da lista começa com a tag <li>.
Os itens desta lista serão marcados com marcadores (circulos pretos) por padrão.

Exemplo:

  • Maça
  • Banana

Podemos mudar os marcadores dos itens não organizados, para isso basta colocarmos a propriedade list-style-type, assim podemos escolher algums modelos, como por exemplo:

Valor Descrição
disc O valor padrão, fica um ponto preto
circle Um circulo com o meio transparente
square Um quadrado
none Um espaço em branco

Lista ordenada

Para utilizarmos este tipo de lista temos que usar primeiro a tag <ol>, e cada item da lista se usa a tag <li>.
Por padrão este iten são marcados por números.

Exemplo:

  1. Casa
  2. Roupa

Alguns tipos de estilos para listas ordenadas:

Valor Descrição
type=1 Default, seria o padrão com os números
type=A A,B,C em maiúsculas
type=a a,b,c em minúsculas
type=I Números romanos em maiúsuclas
type=i Números romanos em minúsculas

Listas de descrição

Uma lista de descrição é uma lista de termos, com uma descrição de cada termo.
A tag <dl> define a lista de descrição, a tag <dt> define o termo (nome) e <dd> descreve cada termo.

Exemplo:

Café
- Bebida preta com cafeina
Leite
- Bebida branca, fria e da vaca

Resumo das Tags

Alinhando listas

Podemos alinhas uma lista dentro de outr lista. Para isso iniciamos uma lista e logo em seguidamos colocamos um outra dentro dela, assim ficamos com sublistas.

Exemplo:

  • Café
  • Água
    1. Com gás
    2. Sem gás

Lista na horizontal

As listas podem ser estilizadas de muitas maneiras no CSS, uma delas é na horizontal, uma maneira muito usada para criar menu de navegação.

Classes - class

Class é usado para manipular elementos de um texto, podendo muda-lo conforme você deseja. Geralmente atribuimos valores a ele através do CSS e podemos atribui-lo a qualquer elemento do HTML.
OBS: o nome da classe diferencia maiúsculas das minúsculas.

Span / div

Podemos atribuir uma class para esse dois elementos, fazendo com que eles mudem suas caracteristicas. Mas a diferença entre o span e div continuam as mesmas.

Sintaxe para Class

Para criamos uma classe, primeiro devemos adicionar um (.), seguido do nome da classe. Logo em seguida, defina as suas propriedade entre as chaves {}:

Exemplo:

        .classe{
            propriedade:valor;
        }

Classes Multiplas

Os elementos em HTML podem ser usados com mais de um classe, para fazermos isso, devemos serpara-los com espaço, como o exemplo a seguir: 

        .city {
            propriedade:valor;
        }

        .main {
            propriedade: valor;
        }

        <h2 class="city main">London</h2>

Diferentes elementos podem conter a mesma classe

Um elemento <p> pode conter a mesma classe que um elemento <h2> e virse-versa.

Uso de Classes em JavaScript

Classes também podem ser usada no JavaScript para realizar certas tarefas para cada elemento específico. JavaScript pode acessar elementos com um nome de classe específico com o metódo getElementsByClassName()

ID

Diferente do class, o atributo id só pode ter em um único elemento no seu proprama, ele é como se fosse uma identidade única. Ele também pode ser usado no JavaScript para estilizar algum elemento do arquivo.

Sintaxe:

Quando queremos criar um id primeiro colocamos hashatg(#), seguindo do nome do id. Em seguida, definimos suas propriedade dentro de chaves {}. E para chama-lo colocamos logo em seguida da tag que você deseja atribuir o valor o seguinte comando, "id='nome_do_id'".
OBS: o id é sensível a letras maiúsculas e minúsculas, e o nome dele deve conter pelo menos um caractere, não podendo começar com um número e não dever conter espaços em branco (espaços, tabulações, etc)

Diferença entre ID e Class

O ID só pode ser usado uma vez, enquanto o class pode ser usado varias vezes por qualquer elemento.

Caminho de Arquivos HTML

Descreve a localização de um arquivo na estrutura de pastas de um site na web.

Exemplos, de arquivos:

Caminho Descrição
<img src="picture.jpg"> O arquivo "picture.jpg" está localizado na mesma pasta da página atual.
<img src="images/picture.jpg"> O arquivo "picture.jpg" está localizado na pasta de imagens da pasta atual.
<img src="/images/picture.jpg"> O arquivo "picture.jpg" está localizado na pasta de imagens raiz da web atual.
<img src="../picture.jpg"> O arquivo "picture.jpg" está localizado na pasta um nível acima da pasta atual.

Caminho de arquivo absolutos

É o caminho da URL completa para um determinado arquivo.

Caminhos de arquivos relativos

Aponta para um arquivo relativo à página atual, por exemplo, o arquivo atual esta na pasta Home e junto com ele tem uma pasta Imagens e você deseja entra para pegar uma imagem, então você vai colocar Imagens/nome_da_foto.

Próximo exemplo:

Quando o arquivo esta em uma página anterior da página atual.
Você vai colocar, ../Imagens/nome_da_foto assim você vai conseguir acessar tranquilamente a foto.

Próximo exemplo:

Quando o arquivo esta localizado na raiz da web atual.
Você vai colocar, /Imagens/nome_da_foto assim conseguirá acessar as fotos.

Boas praticas

Sempre que der use o caminho de arquivo relativo (se possível). Se usarmos algum link para imagens pode acontecer do link para de funcionar e a imagem do site ficar perdida, e praticamente em todos os site usam arquivos em um banco de imagens para evitar este problema.

HTML - Elemento principal

O elemento <head> é um recepiente para os seguintes elementos: <title>, <style>, <meta>, <link>, <script>, e <base>.
Sendo conteiner para metadados(dados sobre dados) e é colocado entre as tags <html> e <body>. Metadados são dados sobre o documento HTML. Os metadados não são exibidos, e normalmente definem o título do documento, conjunto de caracteres, estilos, scripts e outras metainformações.

Elemento <Title>

Define o título do documento. O título deve ser somente texto e é mostrado na barra de título do navegador ou na guia da página. Sendo um item obrigatório no documentos HTML.
O conteúdo do título de uma página é muito importante para a otimização do mecanismo de pesquisa (SEO).

O elemento <title>:

  • Define um título na barra de ferramentas do navegador;
  • Fornece um título para a página quando ela é adicionadas aos favoritos;
  • Exibe um título para a página nos resultados do mecanismo de pesquisa.

Portanto tente tornar o título o mais significativo e preciso possível.

Elemento <style>

Usado para definir informações de estilo para uma única página HTML.

Elemento <link>

Define a relação entre o documento atual e um recurso externo. Mais usada para vincular o arquivo style para à página HTML.

Elemento <meta>

Usado para espeficicar o conjunto de caracteres, a descrição da página, as palavras-chaves, o autor do documento e as configurações da janela de visualização.
Os metadados não serão exibidos na página, mas são usados por navegadores (como exibir conteúdo ou recarregar a página), por mecanismos de pesquisa (palavras-chaves) e outros serviços da web.

Exemplos:

Defina o conjunto de caracteres usado:

<meta charset="UTF0-8">

Defina palavras-chaves para mecanismos de pesquisa:

<meta name="keywords" content="HTML, CSS, JavaScript">

Defina uma descrição de sua página web:

<meta name="description" content="Free Web tutorials">

Defina o autor da página:

<meta name="author" content="Lucas Alexsander">

Atualize o documento a cada 30seg:

<meta http-equiv="refresh" content="30">

Defina a janela de visualização para que seu site tenha uma boa aparência em todos os aplicativos:

<meta name="viewport" content="width=device-width, initial-scale=1.0">


Configurando a janela de visualização

Está janela é a área visível do usuário em uma página da web. Isso varia de acordo com o dispositivo, será menor em um telefone celular do que na tela de um computador.
<meta name="viewport" content="width=device-width, initial-scale=1.0">
Isso fornece ao navegador informações de como controlar as dimensões e escala da página.
A width=device-width define a largura da página para seguir a largura da tela do dispositivo (que irá variar dependendo do dispositivo).
A initial-scale define o zoom inicial quando a página é carregada pela primeira vez pelo navegador.

Elemento usado para definir JavaScript do lado do cliente.

Elemento <base>

Especifica o URL base e / ou destino para todos os URLs relativos a página. Este URL tem que ser o original e nele conseguimos ir para suas subpaginas.
A base deve conter um href e / ou target.

HTML Layout

Os sites geralmente exibem conteúdos em várias colunas (como um jornal ou uma revista).
O HTML possui vários elementos semânticos que definem as diferentes partes de uma página da web:

          
  • <header> - Define um cabeçalho para um documento ou seção.
  • <nav> - Define um conjunto de links de navegação.
  • <section> - Define uma seção em um documento.
  • <article> - Define um conteúdo independente e autocontido.
  • <aside> - Define um conteúdo além do conteúdo (como uma barra lateral).
  • <footer> - Define um rodapé para um documento ou seção.
  • <details> - Define detalhes adicionais que o usuário pode abrir e fechar sob demanda.
  • <summary> Define um título para o <details>.

Tecnicas para criar um Layout

Existem 4 tecnicas, sendo elas:

  • Framework CSS
  • propriedade float CSS
  • CSS flexbox
  • Grade css

Framework

Se você deseja criar um layout rapidamente, você pode usar uma estrutura CSS, como W3C.CSS ou Boostrap.

Float CSS

É comum fazer layouts inteiros da web usando a propriedade float. Float é fácil de aprender, você só precisa lembrar como as propriedades float e clean funcionam. Desvantagem: Os elementos flutuantes estão vinculados ao fluxo do documento, o que pode prejudicar a flexibilidade. Você pode pesquisar mais sobre float no site CSS Float and Clear.
